# KawnHub | النسخة المستقرة (Production)

هذا هو الفرع الرئيسي والمستقر لمشروع KawnHub، وهو الفرع الذي يتم نشره مباشرة على الرابط الرسمي `kawnhub.com`.

حالياً، هذا الفرع يعرض صفحة مؤقتة تشير إلى أن الموقع قيد التطوير.

لمتابعة آخر التطورات والعمل الفعلي على النسخة الجديدة، يرجى الانتقال إلى **فرع `develop`**.
